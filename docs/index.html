Sobre este sitio

Un cuaderno de bitácora de mi primer curso

Esta página web es el proyecto práctico individual de la asignatura Fundamentos de Ingeniería Informática, y está pensada como un mapa de mi paso por primero de carrera: quién soy, qué estoy estudiando, qué dudas tengo todavía y qué cosas me gustan fuera del aula. Todo el sitio está construido únicamente con HTML5 y CSS3 sin frameworks ni scripts de terceros, versionado con Git y publicado a través de GitHub Pages.

Puedes moverte por las distintas secciones desde las pestañas de la cabecera (cada una corresponde literalmente a un archivo .html del repositorio) o desde las tarjetas de aquí abajo.
